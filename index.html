① 💾 저장(Ctrl+S)
② 배포 → 새 배포 → ⚙️ → "웹 앱" 선택
③ 다음 사용자로 실행: 나 / 액세스 권한: 모든 사용자
④ 배포 클릭
⚠️ 권한 승인 안내
A. "액세스 승인" 클릭
B. 구글 계정 선택
C. "Google이 확인하지 않은 앱" → "고급" 클릭
D. "(앱 이름)(으)로 이동" 클릭
E. "허용" 클릭
F. 웹 앱 URL 복사! 📋
